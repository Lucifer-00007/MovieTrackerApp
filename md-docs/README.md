# Documentation

This directory contains comprehensive documentation for the MovieTracker project.

## Structure

### `/info/`
Technical guides and reference documentation:
- **api-integration-guide.md** - Guide for integrating with TMDB and other APIs
- **free-movies-apis.md** - List of available free movie/TV APIs and their features

### `/reports/`
Implementation reports and project summaries:
- **placeholder-image-implementation-report.md** - Details on placeholder image implementation
- **project-summary-report.md** - Overall project status and progress report

### `/summary/`
High-level summaries and overviews (currently empty)

## Purpose

This documentation serves multiple purposes:
- **Developer Onboarding** - Help new developers understand the codebase
- **Technical Reference** - Detailed guides for API integration and implementation patterns
- **Project Tracking** - Reports on feature implementation and project status
- **Decision Records** - Document architectural decisions and their rationale

## Contributing to Documentation

When adding new documentation:
1. Place API guides and technical references in `/info/`
2. Place implementation reports and status updates in `/reports/`
3. Place high-level summaries and overviews in `/summary/`
4. Use clear, descriptive filenames with kebab-case
5. Include a brief description at the top of each document
6. Update this README when adding new categories
